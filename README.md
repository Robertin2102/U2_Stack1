Instrucciones:
Crea un método isPalindrome(String text) que determine si una palabra o frase es palíndroma.
Ignora mayúsculas, espacios y signos.
Ejemplo:
"Anita lava la tina" → true
"Hola" → false

Pista:
Empuja cada letra en la pila y luego saca los elementos para comparar con el texto original.

Entregables:
Url del repo
Capturas de pantalla
Notas:
No usar Stack de java, usa el que de desarrollo en clase.
Usa charAt
